
# fineasy

  
Fineasy is a command-line-interface software to manage cash flow using **Go** with **PostgreSQL** database.
    
## Commands

  ### List  Flows

  By default, this command will list all flows from current month and year.

    fin -lf
    
  ### List  Categories
    fin -lc
    
  ### List  Sources
    fin -lc
    
  ### List  Wallets
    fin -lw
